To use the mViewer tool you need to first install mongoDB. The installation process for mongoDB can be found here. After the installation, make sure that the mongo daemon is running.

For the login process insert the host and port number where the mongod is running. Now either leave the username and password field empty or add a user to the admin database of the mongoDB.

To delete a Database in mViewer, you need to right click on the database name and select the delete option from the context menu. This is depicted in the figure below:

An alert box will appear asking for the confirmation to delete the database. If you click "yes", the database will be deleted and the confirmation of successful database deletion will be shown.
